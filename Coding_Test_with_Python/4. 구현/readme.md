## 구현시 고려할 제약 사항

### 메모리

> 필자는 C++를 사용하고 있기에 C++에 대해 기술되어 있는 내용을 위주로 정리되어 있습니다.

1. **변수 표현 범위**
    - C/C++ 
        - int
            - 4 bytes
            - 범위 : -2,147,483,648 ~ 2,147,483,647
        - long long
            - 8 bytes
            - 범위 : -9,223,372,036,854,775,808 ~ 9,223,372,036,854,775,807
    - python : 자료형을 지정할 필요가 없다.
    - Java
        - BigInteger
            - 가변적 bytes
            - 제한 없음

2. **리스트 크기 제약**

대체로 코딩 테스트에서는 128~512 MB로 메모리를 제한한다. 리스트를 사용하는 경우에 이를 크게 유의해야한다.

- int 자료형 데이터의 개수에 따른 메모리 사용량
    |데이터 개수(리스트의 길이)|메모리 사용량|
    |------|---|
    |1000|약 4KB|
    |1000000|약 4MB|
    |10000000|약 40MB|
